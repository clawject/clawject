What is Clawject and Why‚Äã

Clawject is a Dependency Injection framework for TypeScript with great development experience.

Clawject allows you to not write any boilerplate code that's connecting your classes, use TypeScript type system to define what you need ‚ú®

main.ts
import { Bean, CatContext, ContainerManager } from 'clawject';

interface IBar {}

class Bar implements IBar { /* ... */ }

class Foo {
  constructor(
    private bar: IBar // <-- Clawject will automatically inject implementation of IBar interface
  ) {}

  /* ... */
}

class MyContext extends CatContext {
  foo = Bean(Foo);
  bar = Bean(Bar);
}

ContainerManager.init(MyContext);
